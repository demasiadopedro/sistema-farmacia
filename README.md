# Sobre o projeto

O Sistema surgiu atraves de uma necessidade na Cidade de Santarem, que era o fato de nem todas as ubs possuirem o Horus
que atualmente no Brasil é o software utilizado para a gestao das farmacias da Unidades Básicas de Saúde e com a chegada do Programa de Educação pelo
Trabalho em Saude Digital o desenvolvimento desse sistema foi possivel. A função basicamente dele é fornecer ao gerente da unidade um painel para gerir
o estoque do Programa Hiperdia e dos seus pacientes(que são recorrentes) .
